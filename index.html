Portfolio

Speech Emotion Recognition

Github Repo · Project Overview


The goal of this project was to build a model that predicts the popularity of Airbnb listings in Washington DC and returns recommendations on what to add to the listing to increase its chance of being popular. Popularity for this project is defined as a listing having a 4.8 overall rating or above and at least 60 total reviews.

Data was collected on 8,000 Washington DC listings from Inside Airbnb’s publicly available data and overlayed with venue data on Washington DC neighborhoods from a Foursquare API pull. The listings were cleaned using a combination of feature engineering and natural language processing techniques to prepare for modeling.

The best predictive model and best interpretive model were deployed together in the app for hosts to generate a popularity prediction and recommendations. An Extra Trees Classifier paired with oversampling the minority class was the best model at predicting listing popularity, with an 85% accuracy rate on testing data compared to a 77% baseline. The best interpretive model was Logistic Regression, with an 84% accuracy rate on testing data compared to a 77% baseline, and the coefficients were used to produce listing improvement recommendations.

TSLA Time Series Forecasts

Github Repo
· Project Overview


The goal of this project was to collaborate with data scientist peers and build a model that predicts an adoption outcome for animals taken into the Austin City Animal Center. This shelter is a no-kill shelter, so this model will help with better planning for capacity at the shelter by predicting which animals are likely to be adopted.

Data on 150,000 animal intakes at the shelter were provided from the Austin, Texas government data site. The data was thoroughly cleaned and prepared for modeling. Best performing models for the six different animal intake types were developed using a Gradient Boosting Classifier for each. For the most common intake type of stray animals, the model showed a 83% accuracy score on testing data compared to a 55% baseline score. These models were deployed together in an app that the Austin City Animal Center can use to generate predictions on their new incoming animals.

Wine Recommendation System

Github Repo
· Project Overview


The goal for this project was to build a model using natural language processing techniques to predict whether a given post came from the AskWomen or AskMen subreddit and explore the differences in topics of conversation for each group.

Using Pushshift’s API, 50,000 subreddit posts from both AskWomen and AskMen were collected. The posts were then cleaned and count vectorized to prepare for modeling. The Naïve Bayes model performed the best at predicting which subreddit a given post came from with a 72% accuracy rate on testing data compared to a 50% baseline. The upper limit on accuracy scores remained in the low 70s, as it turns out the topics of conversation in the AskWomen and AskMen are very similar when looking at the most common words and phrases.
